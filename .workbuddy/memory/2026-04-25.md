# 2026-04-25 工作日志

## Bug 修复：utility/scale 蓝图注册失败导致首页 500
- **现象**：访问 /admin/ 首页报 500，`BuildError: Could not build url for endpoint 'utility.utility_list'`
- **根因**：`app/routes/utility.py`（42处）和 `app/routes/scale.py`（16处）中字符串引号被错误转义为 `\'..\'`，导致 Python 语法错误，蓝图模块导入失败
- **修复**：批量替换 `\'` → `'`，两个文件语法验证通过
- **其他文件检查**：`app/__init__.py` 和 `contract_doc_service.py` 中的 `\\` 是合法路径转义，无需修改

## Bug 修复：首页"份合同执行中"数量不对
- **现象**：首页显示的合同数少于合同列表页的13份
- **根因**：`admin_service.py` 查询条件只包含 `Status = N'生效'`，但数据库中合同状态有两种：`生效`(10份) + `有效`(3份)
- **修复**：改为 `Status IN (N'生效', N'有效')`
- **注意**：合同状态字典值有 `生效` 和 `有效`，两者都代表合同正在执行中

## Bug 修复：抄表数据导出图片底部截断
- **现象**：导出的PNG图片底部"合计"行及水印被截断
- **根因**：`.export-container` 使用 `position: fixed; left: -9999px` 离屏定位，html2canvas 对 fixed 元素只渲染视口范围，导致超出视口高度的 tbody/tfoot 被截断
- **修复**：① CSS改为 `position: absolute; left: 0; top: 0; width: 1200px`（z-index: -1 不遮挡页面）② html2canvas 增加 `width`/`height`/`windowWidth`/`windowHeight` 参数，值为 `scrollWidth`/`scrollHeight` ③ setTimeout 从 100ms 增至 300ms
- **二次修复**：html2canvas 方案仍有截断，彻底改为纯 Canvas API 绘制图片（无需 html2canvas 库），删除 export-container CSS 和 CDN 引用
