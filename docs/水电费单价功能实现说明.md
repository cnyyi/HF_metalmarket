# 水电费单价功能实现说明

## 实现时间
2026-04-03

## 实现方案
方案A：在合同关联表中添加单价字段

## 数据库变更

### 1. 添加字段
在以下表中添加 `UnitPrice` 字段：
- `ContractElectricityMeter` 表
- `ContractWaterMeter` 表

字段定义：
```sql
UnitPrice DECIMAL(10,4) DEFAULT 0
```

### 2. 执行迁移
运行迁移脚本：
```bash
sqlcmd -S yyi.myds.me -U sa -P "yyI.123212" -d hf_metalmarket -i "scripts/add_unit_price_to_contract_meter.sql"
```

## 功能说明

### 1. 绑定水电表时设置单价
- 在水电表管理页面（/utility/list），点击"绑定"按钮
- 选择合同后，需要输入单价（元/度 或 元/吨）
- 单价保存到合同关联表中

### 2. 抄表时自动计算金额
- 抄表页面自动从关联表获取单价
- 根据用量和单价自动计算金额
- 金额 = 用量 × 单价

### 3. 抄表记录保存单价
- 抄表记录表（UtilityReading）中保存单价
- 便于后续查询和历史追溯

## 业务优势

1. **灵活性高**：不同合同可以有不同的水电费单价
2. **数据完整**：抄表记录保存单价，便于历史追溯
3. **自动计算**：系统自动计算金额，减少人工错误
4. **符合业务**：符合实际业务场景（不同商户可能有不同的单价协议）

## 注意事项

1. 绑定水电表时必须输入单价，且单价必须大于0
2. 如果合同关联表中没有单价记录，默认为0
3. 抄表记录中的单价来自绑定时的设置，不会自动更新
4. 如需修改单价，需要解绑后重新绑定
