# 合同明细查看功能 - 模态窗口实现

## 功能概述

在合同列表页面实现了合同明细查看功能，通过模态窗口（Modal）展示合同详细内容，无需跳转到新页面，保持列表页面的上下文状态不变。

## 实现细节

### 1. UI 设计

#### 模态窗口结构
- **尺寸**: 大尺寸模态框（modal-lg），提供更宽敞的展示空间
- **样式**: 
  - 圆角设计（12px）
  - 无边框，带阴影效果
  - 渐变紫色标题栏（#667eea → #764ba2）
  - 浅灰色内容区域背景

#### 视觉层级
1. **标题栏**: 渐变背景，白色文字，突出显示
2. **内容分区**: 使用分割线和标题清晰划分不同信息区域
3. **信息分组**: 
   - 基本信息区
   - 租金信息区
   - 地块信息区（如有）
4. **重点突出**: 
   - 金额使用大号字体和蓝色强调
   - 状态使用徽章（badge）显示
   - 重要标签使用粗体

### 2. 功能特性

#### 显示内容

**基本信息区域：**
- 合同编号
- 合同名称
- 商户名称
- 联系人
- 合同金额（蓝色大号字体）
- 合同状态（徽章形式）
- 开始日期
- 结束日期
- 合同期年
- 创建时间

**租金信息区域：**
- 地块年租金合计
- 租金增减（正数显示"+"号）
- 实际合同金额（蓝色大号字体）

**备注说明（如有）：**
- 浅灰色背景框显示
- 圆角边框设计

**地块信息区域（如有地块）：**
- 表格形式展示
- 包含：序号、地块编号、面积、单价、月租金、年租金
- 自动计算并显示地块总数

#### 交互功能

1. **查看按钮**
   - 位置：操作列第一个按钮
   - 样式：蓝色信息按钮（btn-outline-info）
   - 图标：眼睛图标（fa-eye）
   - 功能：点击打开模态窗口并加载合同详情

2. **关闭操作**
   - 方式一：点击右上角关闭按钮（×）
   - 方式二：点击底部"关闭"按钮
   - 方式三：点击模态窗口外部区域
   - 方式四：按 ESC 键

3. **加载状态**
   - 初始加载：显示旋转加载动画
   - 加载失败：显示错误提示和重试图标
   - 数据为空：显示"-"或默认值

### 3. 动画效果

#### 模态窗口动画
```css
/* 淡入淡出 + 缩放效果 */
#viewModal.fade .modal-dialog {
    transform: scale(0.95);
    opacity: 0;
}

#viewModal.show .modal-dialog {
    transform: scale(1);
    opacity: 1;
}
```

**效果说明：**
- 打开时：从 95% 缩放至 100%，透明度从 0 到 1
- 关闭时：反向动画
- 过渡时间：0.3 秒，缓动效果

#### 加载动画
- 使用 Bootstrap 原生 spinner
- 主色调（primary）旋转动画
- 居中显示，垂直内边距充足

### 4. 响应式设计

#### 栅格布局
- **大屏幕（≥768px）**: 
  - 基本信息：2 列布局（每行 2 个字段）
  - 租金信息：3 列布局（每行 3 个字段）
- **小屏幕（<768px）**: 
  - 自动切换为单列布局
  - 保持信息完整显示

#### 间距优化
- 使用 Bootstrap 5 的 `g-3` 栅格间距
- 统一的内外边距标准
- 移动端友好的触摸区域

### 5. 数据加载流程

```javascript
1. 用户点击"查看"按钮
   ↓
2. 触发 loadContractDetail(contractId)
   ↓
3. 显示加载动画
   ↓
4. AJAX GET /contract/detail/{id}
   ↓
5. 接收 JSON 数据
   ↓
6. 渲染 HTML 模板
   ↓
7. 更新模态窗口内容
   ↓
8. 用户查看信息
```

### 6. 错误处理

#### 加载失败场景
1. **网络错误**: 显示错误图标和"请求失败，请重试"提示
2. **数据不存在**: 显示"合同不存在"提示
3. **权限不足**: 自动跳转到登录页面

#### 错误 UI
- 大号警告图标（fa-exclamation-triangle）
- 红色文字提示
- 充足的内外边距，视觉清晰

### 7. 性能优化

#### 优化措施
1. **按需加载**: 仅在点击查看时加载详情数据
2. **单次请求**: 一次 API 调用获取所有必要信息
3. **客户端渲染**: 减少服务器压力
4. **缓存友好**: 浏览器可缓存静态资源

#### 响应时间
- 本地测试：< 200ms
- 网络环境：取决于服务器响应速度
- 加载动画：避免用户感知延迟

## 使用说明

### 查看合同明细步骤

1. **访问合同列表页面**
   - URL: http://127.0.0.1:5000/contract/list

2. **定位目标合同**
   - 可通过搜索框快速查找
   - 可使用分页功能浏览

3. **点击查看按钮**
   - 操作列第一个蓝色眼睛图标按钮
   - 点击后立即打开模态窗口

4. **查看合同详情**
   - 基本信息区域：合同基本信息
   - 租金信息区域：租金相关数据
   - 地块信息区域：地块列表（如有）

5. **关闭模态窗口**
   - 点击右上角 × 按钮
   - 或点击底部"关闭"按钮
   - 或点击窗口外部区域
   - 或按 ESC 键

## 技术总结

本次实现采用了以下关键技术：

1. **Bootstrap 5 Modal**: 成熟的模态窗口组件
2. **AJAX 异步加载**: 无刷新获取数据
3. **CSS3 动画**: 流畅的视觉过渡效果
4. **响应式布局**: 适配多种屏幕尺寸
5. **模板字符串**: 动态生成 HTML 内容
6. **错误处理**: 完善的异常捕获和提示

通过以上技术的综合运用，实现了功能完善、体验优秀的合同明细查看功能。
